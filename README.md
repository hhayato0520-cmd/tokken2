# 論文の骨組み: 大規模言語モデルを用いた制約考慮型グループ料理レシピ推薦システムの提案
- アブストラクト: 
  - 近年，料理レシピ推薦に関する研究が盛んに行われているが，多くは個人を対象とした推薦を前提としている．
  - 一方で，家庭や友人グループなど複数人で食事を決定する場面では，参加者ごとに異なる嗜好や食事制約が存在するため，全員が満足できる料理を選択することは容易ではない．
  - 特に，アレルギーや苦手食材，辛味の許容度，予算などの制約が混在する場合には，個人向け推薦手法をそのまま適用することは困難である．
  - 本研究では，複数ユーザの嗜好および食事制約を考慮したグループ向け料理レシピ推薦システムを提案する．
  - 提案手法では，ユーザが自然言語で入力した食の好みや制約条件から，大規模言語モデル（LLM）を用いてユーザプロファイルを生成する．
  - また，楽天レシピから取得したレシピ情報に加え，レビュー情報や食材価格情報を利用して各レシピの特徴を構築する
  - さらに，グループ全体の満足度，ユーザ間の意見の不一致度，制約違反の有無，およびレシピの実用性を統合した評価関数を定義し，そのスコアに基づいて推薦順位を決定する．
