# 論文の骨組み: 大規模言語モデルを用いた制約考慮型グループ料理レシピ推薦システムの提案
- アブストラクト: 
  - 近年，料理レシピ推薦に関する研究が盛んに行われているが，多くは個人を対象とした推薦を前提としている．
  - 一方で，家庭や友人グループなど複数人で食事を決定する場面では，参加者ごとに異なる嗜好や食事制約が存在するため，全員が満足できる料理を選択することは容易ではない．
  - 特に，アレルギーや苦手食材，辛味の許容度，予算などの制約が混在する場合には，個人向け推薦手法をそのまま適用することは困難である．
  - 本研究では，複数ユーザの嗜好および食事制約を考慮したグループ向け料理レシピ推薦システムを提案する．
  - 提案手法では，ユーザが自然言語で入力した食の好みや制約条件から，大規模言語モデル（LLM）を用いてユーザプロファイルを生成する．
  - また，楽天レシピから取得したレシピ情報に加え，レビュー情報や食材価格情報を利用して各レシピの特徴を構築する
  - さらに，グループ全体の満足度，ユーザ間の意見の不一致度，制約違反の有無，およびレシピの実用性を統合した評価関数を定義し，そのスコアに基づいて推薦順位を決定する．

- キーワード:
  - グループ推薦,レシピ推薦,セレンディピティ,LLM,嗜好解析,公平性
    
1. はじめに

   - P1：研究背景
     - 家族や友人など複数人で食事を決定する場面が日常的に存在する．
     - レシピサイトの普及により選択肢が増大している．
     - 多数の候補から全員が満足する料理を選択することは容易ではない．

   - P2：従来研究の課題
     - 個人向け料理推薦研究が主流である．
     - グループ推薦研究は映画や音楽分野が中心である．
     - 料理推薦への適用事例は少ない．

   - P3：グループ料理推薦の課題
     - 嗜好の衝突が発生する．
     - アレルギーや予算などの制約が存在する．
     - 平均化のみでは不公平な推薦になる可能性がある．

   - P4：セレンディピティの必要性
     - 従来推薦は既知の嗜好に偏る．
     - 新しい料理との出会いが失われる．
     - 推薦システムに多様性が求められている．

   - P5：研究目的
     - グループ満足度と公平性を考慮した料理推薦手法を提案する．
     - セレンディピティを導入した推薦を実現する．

   - P6：研究アプローチ
     - LLMによる嗜好解析
     - 制約考慮型推薦
     - セレンディピティスコア導入

   - P7：本研究の貢献
     - グループ料理推薦のための統合評価関数の提案
     - セレンディピティ導入手法の提案
     - 実コストを考慮した推薦システムの構築

   - P8：論文構成
